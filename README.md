# Bash

## Loops in bash

Source: https://www.cyberciti.biz/faq/bash-for-loop/

```
#!/bin/bash
for i in {1..5}
do
   echo "Welcome $i times"
done
```

## Operations in bash

For simple operations:

```
num=$(($num1 + $num2))
```

**Warning: do not forget both brackets**
